Challenge: Análisis de Evasión de Clientes (Churn) – Alura One
Descripción

Este proyecto analiza los factores que influyen en la evasión de clientes (Churn) de una empresa de telecomunicaciones. Se utiliza un dataset en formato JSON con información de clientes, servicios contratados y facturación.

Objetivo

Identificar patrones y características que ayuden a reducir la pérdida de clientes mediante análisis exploratorio y visualizaciones.

Contenido del Notebook

El notebook está organizado en cuatro bloques:

Extracción: carga de los datos desde el JSON en GitHub.

Transformación: limpieza de datos, aplanado de columnas tipo dict, conversión de categorías y tipos de datos.

Carga y Análisis: análisis exploratorio con gráficos y correlaciones para entender el Churn.

Informe Final: resumen de hallazgos, conclusiones y recomendaciones estratégicas.

Herramientas utilizadas

Python 3

Pandas

Matplotlib

Seaborn

Resultado

Se identificaron variables clave que afectan la evasión: antigüedad del cliente, tipo de contrato, servicios adicionales contratados y facturación mensual. El análisis proporciona insights accionables para retener clientes y reducir el Churn.
